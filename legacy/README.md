# Legacy Benchmarks

This directory contains legacy benchmark implementations that were migrated from the main OpenHands repository. These benchmarks are preserved for historical reference but are not actively maintained.

## Note

These benchmarks are excluded from linting and type-checking. If you need to use any of these benchmarks, please consider:

1. Porting them to the main `benchmarks/` directory with proper maintenance
2. Using them as-is with the understanding that they may require updates
